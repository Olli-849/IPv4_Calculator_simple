# IPv4_Calculator_simple

Ein einfacher Rechner für IPv4-Adressen.

Man gibt IPv4-Adresse und die Subnetz-Maske ein und ermittelt werden:
- CIDR
- Netz-ID
- Broadcast
- nummerischer Host-Anteil
